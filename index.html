      Facilitator notes (don't screen-share this part)

      The loop per round: set the scene (right panel) → state the stakes out loud → participant makes their attempt in English → roll → narrate the consequence to match the number → give a quick language upgrade → next beat. Aim for 3 beats: setup, complication, resolution.

      Keep it moving

      • The die decides the fiction; you decide the language. Don't roll to judge English.
      • On a low roll, make failure interesting — a new complication, not a dead end.
      • Stay in your character's voice; give one memorable trait so participants have something to react to.
      • End every beat on a question or pressure so the participant has something to do.

      Language

      • Catch emergent language; bank the best lines in the Capture box live.
      • B1: let them lean on the phrase chips. C1: hide the chips, push tone, concision, audience.
      • Run the same function all game — one clean function beats five messy ones.

      Reuse / share

      • No login, no install — open the link or run locally. Shareable with other trainers.
      • Realistic ↔ Fantasy toggle reskins the same mission; B1/C1 reskins the support.
      • A+ later: hand the GM seat (this panel) to a confident participant with a Copilot/Gemini prompt; the timer forces a close.
